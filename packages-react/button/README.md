# @pittorica/button-react

The `Button` component.

## Installation

```bash
npm install @pittorica/button-react
```

## Usage

```jsx
import { Button } from '@pittorica/button-react';

function MyComponent() {
  return (
    <Button variant="filled" color="indigo" onClick={() => alert('Clicked!')}>
      Click Me
    </Button>
  );
}
```
