# @pittorica/em-react

The `Em` component.

## Installation

```bash
npm install @pittorica/em-react
```

## Usage

```jsx
import { Em } from '@pittorica/em-react';

function MyComponent() {
  return (
    <p>
      This is a <Em>very important</Em> message.
    </p>
  );
}
```
