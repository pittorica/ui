# @pittorica/heading-react

The `Heading` component.

## Installation

```bash
npm install @pittorica/heading-react
```

## Usage

```jsx
import { Heading } from '@pittorica/heading-react';

function MyPage() {
  return (
    <>
      <Heading as="h1" size="9">
        Main Title
      </Heading>
      <Heading as="h2" size="7">
        Subtitle
      </Heading>
    </>
  );
}
```
