# @pittorica/link-react

The `Link` component.

## Installation

```bash
npm install @pittorica/link-react
```

## Usage

```jsx
import { Link } from '@pittorica/link-react';

function MyLink() {
  return (
    <Link href="https://pittorica.dcdavidev.me" target="_blank">
      Documentation
    </Link>
  );
}
```
