# @pittorica/quote-react

The `Quote` component.

## Installation

```bash
npm install @pittorica/quote-react
```

## Usage

```jsx
import { Quote } from '@pittorica/quote-react';

function MyText() {
  return (
    <p>
      As they say, <Quote>less is more</Quote>.
    </p>
  );
}
```
