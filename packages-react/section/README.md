# @pittorica/section-react

The `Section` component.

## Installation

```bash
npm install @pittorica/section-react
```

## Usage

```jsx
import { Section, Heading, Text } from '@pittorica/react';

function MyPage() {
  return (
    <Section size="3">
      <Heading>About Us</Heading>
      <Text>Modular and accessible design system.</Text>
    </Section>
  );
}
```
