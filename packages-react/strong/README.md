# @pittorica/strong-react

The `Strong` component.

## Installation

```bash
npm install @pittorica/strong-react
```

## Usage

```jsx
import { Strong } from '@pittorica/strong-react';

function MyComponent() {
  return (
    <p>
      This is <Strong>very important</Strong> information.
    </p>
  );
}
```
