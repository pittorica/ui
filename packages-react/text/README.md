# @pittorica/text-react

The `Text` component.

## Installation

```bash
npm install @pittorica/text-react
```

## Usage

```jsx
import { Text } from '@pittorica/text-react';

function MyComponent() {
  return (
    <Text size="3" weight="bold" color="indigo">
      Hello World
    </Text>
  );
}
```
